# ✨ Gemini Query Assistant

A modern, interactive chatbot web application built with Streamlit and Google's Gemini AI. Ask questions, get intelligent responses, and enjoy a seamless conversational experience.

## 🌟 Features

- 💬 **Real-time Chat Interface** - Clean, user-friendly chat UI
- 🤖 **Powered by Gemini** - Leverages Google's advanced AI model (Gemini Flash)
- 💭 **Conversation Memory** - Maintains context across multiple messages
- 🎨 **Markdown Support** - Displays formatted responses with code blocks, lists, and more
- 🔒 **Secure API Key Management** - Uses environment variables for sensitive data
- ⚡ **Fast Responses** - Optimized for quick AI interactions

##  Prerequisites

- Python 3.8 or higher
- Google Gemini API Key (free at https://aistudio.google.com/app/apikey)
